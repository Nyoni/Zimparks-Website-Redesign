# ZimParks Responsive Mega Menu UI System

## What is included
- `zimparks_responsive_mega_menu_ui_system.svg`: editable Sketch import file with desktop, tablet, mobile, foundations, and prototype map.
- `index.html`, `styles.css`, `script.js`: responsive interactive prototype similar to the provided MRP mega menu reference.
- `design-tokens.json`: colors, typography, breakpoints, and Sketch setup notes.

## How to view and edit in Sketch
1. Unzip this package.
2. Open Sketch.
3. Use **File > Insert > From File...**.
4. Select `zimparks_responsive_mega_menu_ui_system.svg`.
5. Sketch will import the artboards as editable SVG layers.
6. Select repeated components and use **Create Symbol** for nav items, buttons, sidebars, columns, and visual cards.

## How to test the interactive prototype
1. Open `index.html` in a browser.
2. On desktop, hover navigation items to change mega menu content.
3. On mobile width, click `☰ MENU` to open the accordion drawer.
